# QuakeZone - Soluvas Workspace

## Development Setup

Requirements:

* Visual Studio Code
* Node.js 12
* yarn v1
* MongoDB

Steps:

1. Open this in Visual Studio Code or Gitpod
2. Run: `yarn`
3. In first terminal:

   ```bash
   cd packages/strapi
   strapi develop
   ```

4. In second terminal:

   ```bash
   cd packages/pwa
   yarn start
   ```
